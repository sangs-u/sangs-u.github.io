NPC 실수 확률 계산식
mistakeRate = (fatigue/100 × 0.3) + ((1 - skill) × 0.2) + languageGap × 0.4
// fatigue: 피로도 0~100 | skill: 숙련도 0~1 | languageGap: 언어장벽 0~1

TBM 실시 시: mistakeRate × 0.6 // 40% 감소
시범 보임 시: mistakeRate × 0.5 // 50% 감소
그림TBM 시: languageGap × 0.3 // 외국인 오해 70% 감소
행동 스크립트 구조 (공통 + 공종별)
스크립트 = 공통 행동 70% + 공종 전용 30% → 조합으로 경우의 수 극대화
